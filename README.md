# 带玩平台触手API-英雄联盟国服数据接口
1. API主要针对访问量过大的应用场景，独享带宽和资源，启动触手版API服务器程序（DaiWan.Tentacle.exe），此时这个电脑就成为一台API Server。
2. 默认调用URL：http://localhost:4444/{API}
3. 可自行调整为本地服务器的IP地址，您的应用可部署多台API Server自行进行负载均衡处理。
